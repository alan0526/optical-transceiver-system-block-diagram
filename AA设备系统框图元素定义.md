# 8通道双Lens OSPF光模块 AA设备 — 系统框图元素定义

> 用途：为DOE实验文档提供统一的部件名称引用，每个DOE任务可直接写明操作的是框图中哪个部件。

---

## 一、光模块DUT（被测器件）

### 1.1 TOSA（发射端，按光路顺序）

> **光路**：LD → Laser Lens → Isolator → Fiber Lens → Tx MPO接口
> **背光监控**：LD 背向出光 → MPD
> **电驱动**：DSP → LD（调制信号）
> **温控**：TEC（位于LD下方）

| 编号 | 名称 | 说明 |
|------|------|------|
| DUT-T-LD | **LD** | 激光器芯片，光路起点 |
| DUT-T-MPD | **MPD** | 背光监控光电二极管，接收LD背向光 |
| DUT-T-TEC | **TEC** | 热电制冷器，控制LD温度 |
| DUT-T-LL | **Laser Lens** | 第一透镜，LD出光准直，AA操作对象 |
| DUT-T-ISO | **Isolator** | 光隔离器，防止回波反射干扰LD |
| DUT-T-FL | **Fiber Lens** | 第二透镜，将准直光聚焦入光纤，AA操作对象 |
| DUT-T-MPO | **Tx MPO接口** | 发射端光输出连接器 |

### 1.2 ROSA（接收端，按光路顺序）

> **光路**：Rx MPO接口 → PD → TIA
> **电信号**：TIA → DSP（信号处理与恢复）

| 编号 | 名称 | 说明 |
|------|------|------|
| DUT-R-MPO | **Rx MPO接口** | 接收端光输入连接器，光路入口 |
| DUT-R-FL | **Rx Fiber Lens** | 将光纤中的光聚焦到光电探测器，AA操作对象 |
| DUT-R-PD | **PD** | 光电探测器，将光信号转为电流 |
| DUT-R-TIA | **TIA** | 跨阻放大器，将PD电流转为电压，输出至DSP |

### 1.3 公用

| 编号 | 名称 | 说明 |
|------|------|------|
| DUT-C-DSP | **DSP** | 数字信号处理器，全模块仅一颗；Tx方向驱动LD调制，Rx方向接收TIA信号恢复数据 |
| DUT-C-ELEC | **电气接口** | 供电 + DDM通信（I2C/MDIO） |

## 二、光路仪器与光纤链路

### 2.1 Tx端AA光路（DUT Tx → 测量仪器）

> **信号流向**：DUT Tx MPO → Tx测量光纤 → 光功率计
> **波前检测**：DUT Tx 光路 → 90度转镜 → 波前传感器

| 编号 | 名称 | 说明 |
|------|------|------|
| OPT-Tx-FIB | **Tx测量光纤** | 一端MPO接DUT Tx MPO，另一端FC接光功率计光输入口 |
| OPT-Tx-OPM | **光功率计 (OPM)** | 测量Tx端耦合输出光功率，含FC光输入口与模拟输出口 |
| OPT-Tx-MIR | **90度转镜** | （可选）置于Fiber Lens位置，将Tx光路折转90°送入波前传感器，避让光模块内部拥挤结构 |
| OPT-Tx-WFS | **波前传感器** | （可选）经90度转镜接收Tx光信号，反馈泽尼克系数用于TOSA调平；可用功率搜索方式替代 |

### 2.2 Rx端AA光路（标准光源 → DUT Rx）

> **信号流向**：标准光源 → Rx输入光纤 → 光衰减器（可选）→ DUT Rx MPO
> **锁定检测**：锁相放大器配合调制标准光源提升SNR

| 编号 | 名称 | 说明 |
|------|------|------|
| OPT-Rx-SRC | **标准光源** | 为Rx端AA提供稳定可调光源（波长/功率可控），FC输出 |
| OPT-Rx-FIB | **Rx输入光纤** | 一端FC接标准光源输出，另一端MPO接DUT Rx MPO |
| OPT-Rx-VOA | **光衰减器 (VOA)** | 插入Rx光路，动态调节进入DUT Rx的光功率（可选） |
| OPT-Rx-LIA | **锁相放大器 (LIA)** | 配合调制标准光源，提升ROSA端信噪比 |

## 三、电子链路与信号采集

### 3.1 Tx端功率采样链路

> **信号流向**：OPM模拟输出口 → 同轴/差分线 → 自制ADC → 数字接口 → PC

| 编号 | 名称 | 说明 |
|------|------|------|
| ELEC-Tx-OPM | **OPM模拟输出口** | 光功率计将测得的光功率转换为模拟电压输出 |
| ELEC-Tx-ADC | **自制ADC采样模块** | 对OPM模拟信号做抗混叠滤波与模数转换；含外部触发输入口，接收运控IO触发脉冲并锁存该时刻光功率值 |
| ELEC-Tx-PC | **PC数据采集** | ADC数字量通过USB/PCIe/以太网读入PC |

### 3.2 DUT电气链路（按电流流向排序）

> **供电回路**：高精度程控电源 → 供电线缆 → 顶针触点 → 光模块电口焊盘 → DUT
> **Tx信号链路**：DSP → LD调制驱动（DUT内部）
> **Rx信号链路**：TIA → DSP（DUT内部）
> **监控信号回路**：光模块电口焊盘 → 顶针触点 → DUT通信链路 → PC

| 编号 | 名称 | 说明 |
|------|------|------|
| ELEC-DUT-PSU | **高精度程控电源** | 供电起点，可编程输出电压/电流，满足纹波与瞬态响应要求 |
| ELEC-DUT-CBL | **供电线缆** | 程控电源输出至顶针载体的连接线缆（含供电线与地线） |
| ELEC-DUT-PIN | **顶针触点** | 与光模块电口焊盘弹性接触，供电与信号共用导通 |
| ELEC-DUT-TEC | **TEC驱动** | DUT内部TEC的温控驱动，经顶针触点供电 |
| ELEC-DUT-COM | **DUT通信链路** | I2C/MDIO 经顶针触点读取DDM数据（LD功率、温度、电压、偏置电流），信号回传至PC |

## 四、运控系统（飞行扫描位置-功率同步）

> **同步原理**：运控轴运动到预设位置时，通过外部IO输出触发脉冲 → ADC锁存该时刻的OPM光功率值
> **信号流向**：运控轴编码器 → 位置比较 → 外部IO触发 → ADC外部触发输入 → 锁存光功率值 → PC读取

| 编号 | 名称 | 说明 |
|------|------|------|
| MOT-AXIS | **运控轴** | 多轴运动平台（X/Y/Z/θx/θy/θz），携带编码器位置反馈 |
| MOT-TRIG | **位置触发IO** | 运控控制器在预设位置输出电平跳变，作为ADC外部触发信号 |
| MOT-TCBL | **触发信号线缆** | 运控IO输出至ADC外部触发输入口的连接线缆 |

## 五、编号体系总览

| 前缀 | 含义 | 范围 |
|------|------|------|
| DUT-T-xx | 光模块TOSA | LD、MPD、TEC、Laser Lens(LL)、Isolator(ISO)、Fiber Lens(FL)、Tx MPO |
| DUT-R-xx | 光模块ROSA | Rx MPO、Lens、PD、TIA |
| DUT-C-xx | 光模块公用 | DSP、电气接口(ELEC) |
| OPT-Tx-xx | Tx端光路 | Tx测量光纤(FIB)、光功率计(OPM)、90度转镜(MIR)、波前传感器(WFS) |
| OPT-Rx-xx | Rx端光路 | 标准光源(SRC)、Rx输入光纤(FIB)、光衰减器(VOA)、锁相放大器(LIA) |
| ELEC-Tx-xx | Tx电子链路 | OPM模拟输出(OPM)、ADC采样模块(ADC)、PC数据采集(PC) |
| ELEC-DUT-xx | DUT电气链路 | 程控电源(PSU)、供电线缆(CBL)、顶针触点(PIN)、通信(COM) |
| FIX-TEC | 治具热电制冷器 | 光模块治具的制冷器 |
| MOT-xx | 运控系统 | 运控轴(AXIS)、位置触发IO(TRIG)、触发信号线缆(TCBL) |
